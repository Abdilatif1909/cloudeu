# Teacher Guide

## Course Management
- Teachers can create and manage only their assigned courses.
- Teachers can upload lectures, practical materials, videos, resources, glossary entries, and quizzes.
- Teachers can import quizzes using the JSON importer from the Quiz admin or REST endpoint.

## Content Workflow
1. Create course lessons.
2. Upload lecture PDF, cover image, and description.
3. Upload practice PDFs, archives, and source files.
4. Add YouTube videos; the system extracts video IDs and thumbnails.
5. Import or create quizzes and define pass score/timer/randomization.
6. Publish content when ready.

## Statistics
- Use teacher analytics API for active students, average scores, popular lectures/videos, downloads, and quiz statistics.
