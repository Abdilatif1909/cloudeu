# Student Guide

## Learning
- Students see enrolled courses in their course list.
- Lecture pages include PDF preview, reader controls, progress, notes, and bookmarks.
- Video pages support resume position and completion tracking.
- Practice pages support file downloads and completion marking.
- Dashboard shows completion, study time, streak, recent videos/PDFs, notifications, and weekly activity.

## Certificates
- A certificate can be generated when course completion is 100% and quiz average is at least 70%.
- Certificates include student name, course name, issue date, certificate ID, and verification code.
